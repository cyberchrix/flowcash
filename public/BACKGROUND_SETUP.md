# Configuration du fond d'écran

Pour utiliser l'image de fond en mode dark :

1. Placez votre image de gradient dans le dossier `public/`
2. Nommez-la `background-gradient.jpg` (ou `.png` selon le format)
3. L'image sera automatiquement utilisée comme fond en mode dark

L'image sera adaptée à la taille de l'écran avec `background-size: cover` et restera fixe lors du scroll.
